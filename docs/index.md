# airbyte Provider

airbyte-api: Programmatically control Airbyte Cloud, OSS & Enterprise.

-> New to the Airbyte Terraform provider? See the [Getting Started](guides/getting_started) guide.

-> **Upgrading to 1.0?** Typed connector-specific resources (e.g., `airbyte_source_postgres`) are deprecated in 1.0 and targeted for removal in 1.1. See the [Migrating to 1.0](guides/v1_migration_guide) guide to move to the generic `airbyte_source` / `airbyte_destination` resources with the [`airbyte_connector_configuration`](data-sources/connector_configuration.md) data source for type-validated configuration.

## Example Usage

```terraform
terraform {
  required_providers {
    airbyte = {
      source  = "airbytehq/airbyte"
      version = "1.0.0-rc7"
    }
  }
}

provider "airbyte" {
  server_url = "..." # Optional
}
```
